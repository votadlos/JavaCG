JacORB - Eclipse Integration mini HOW-TO

This document describes how to set up a project in eclipse to step through, debug, or browse the source of JacORB.

  1. A dependency of this HOWTO is the prior installation of Ant, and the setting of the ANT_HOME environment variable. Don't open eclipse yet.
  2. Download and unpack the desired version of JacORB. For purposes of this document, I have unpacked JacORB into C:\jacorb_2_2_1.
  3. Set your JACORB_HOME environment variable. 
     
				set JACORB_HOME=C:\jacorb_2_2_1
  4. From the JACORB_HOME directory, execute ant.
  5. Once ant has completed successfully, open eclipse.
  6. Create a new Java project, and use JACORB_HOME as the project's root.
  7. In the Java Build Path screen, either while setting up the project or by going into the project properties after creating the project, you must include the following source directories:
    • JACORB_HOME/src
    • JACORB_HOME/src/generated
  8. Now you must exclude certain other directories (we will exclude those parts of JacORB that do not build cleanly under eclipse in this simple setup. Depending on your needs, you may want to include some of them and work through their dependencies yourself). Double click on "Excluded ..." beneath JACORB_HOME/src. Exclude the following directories:
    • generated
    • HTTPClient
    • omg-02-01-02
    • org/jacorb/notification
    • org/jacorb/orb/http
    • org/jacorb/security/level2
    • org/jacorb/ssl
    • org/jacorb/util
  9. Add the required jar files on the build path/Libraries window. These are all located in the JACORB_HOME/lib directory, plus ant.jar from your ant distribution.
  10. Let eclipse build the project.
